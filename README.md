# FamilyBites

FamilyBites is a private family food hub.

## Version 1 includes

- Premium landing page
- Netflix-style family profile selection
- Personal dashboard
- Snap Food form
- Favorite restaurants
- Weekly AI recap mockup
- Mobile responsive layout

## Upload to GitHub

Upload these files to your `kokolo2020/familybites` repository:

- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `sql/schema.sql`

## Deploy to Netlify

After the files are in GitHub:

1. Go to Netlify
2. Add new site
3. Import from GitHub
4. Choose `kokolo2020/familybites`
5. Deploy
